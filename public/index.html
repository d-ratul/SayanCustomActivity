Please make sure that the data extension you select contains a column by name of HOLIDAYDATE of type Date. If DATA EXTENSION NAME WHERE HOLIDAYS ARE STORED is left blank, emails will be sent on days selected in left panel. If you provide a value of data extension, the emails will be sent on specific days selected and system will check if the holiday list contains today's date or not. Think of it as a union of two lists i.e list of days and list of holidays.
